# Grokking Algorithms – Study Repository

## 📘 About

This repository contains my notes, explanations, and code implementations based on the book:

**"Grokking Algorithms: An Illustrated Guide for Programmers and Other Curious People" – Aditya Y. Bhargava**

The goal of this project is to strengthen my understanding of fundamental algorithms and data structures by:

- Rewriting concepts in my own words
- Implementing each algorithm from scratch
- Solving exercises
- Practicing problem-solving skills
